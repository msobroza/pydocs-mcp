# Security Policy

## Supported versions

The latest minor release on PyPI is supported. Older minor releases receive
only critical security fixes for 6 months after the next minor ships.

## Reporting a vulnerability

Please **do not** open public GitHub issues for security reports.

Instead, use GitHub's private vulnerability reporting:

1. Go to https://github.com/msobroza/pydocs-mcp/security/advisories
2. Click "Report a vulnerability"
3. Provide repro steps, affected versions, and impact assessment

We aim to:

- Acknowledge within 72 hours
- Confirm reproduction within 7 days
- Ship a fix within 30 days for high/critical severity

You will be credited in the GitHub Security Advisory and in `CHANGELOG.md`
unless you prefer anonymity.
